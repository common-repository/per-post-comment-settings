You can change comment settings post by post on WP 2.7 or greater.

== Description ==

When you want to apply different comment settings like comments per page, order of comments, .... for specific post or page, this plugin would help you do what you want.

Some requirements

* Your theme should compatible with wordpress 2.7 comments system.
* Octto wrote about <a href="http://ottodestruct.com/blog/2008/09/29/wordpress-27-comments-enhancements/">wordpress 2.7 comments enhancement</a>.
* In short, your `comments.php` must use `wp_list_comments()` function.

== Installation ==

1. Upload `per-post-comment-settings.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
